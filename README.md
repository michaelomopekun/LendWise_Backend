# Hello World TypeScript Project

This is a simple TypeScript project that demonstrates a basic "Hello World" application.

## Project Structure

```
hello-world-ts
├── src
│   └── index.ts
├── package.json
├── tsconfig.json
└── README.md
```

## Getting Started

To get started with this project, follow the instructions below.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository or download the project files.
2. Navigate to the project directory:
   ```
   cd hello-world-ts
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the Application

To run the application, use the following command:
```
npm start
```

This will compile the TypeScript code and execute the `index.ts` file, which logs "Hello, World!" to the console.
